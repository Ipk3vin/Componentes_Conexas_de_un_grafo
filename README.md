# Componentes Conexas de un Grafo — Web App

Esta es una aplicación web moderna desarrollada con **HTML, CSS y JavaScript** que permite:

- Generar grafos aleatorios o manuales
- Dibujar nodos y aristas dirigidas
- Construir la matriz de adyacencia
- Calcular las **componentes conexas**
- Mostrar cada componente como un subgrafo independiente
- Visualizar paso a paso el proceso de análisis

La interfaz está diseñada con un estilo moderno tipo dashboard y utiliza la tipografía **Poppins**.

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript (Canvas API)
- Google Fonts (Poppins)
